# UU 22 Tahun 2009 Pasal 114

## Bunyi Pasal

> Pada perlintasan sebidang antara jalur kereta api dan Jalan, Pengemudi Kendaraan wajib: a. berhenti ketika sinyal sudah berbunyi, palang pintu kereta api sudah mulai ditutup, dan/atau ada isyarat lain; b. mendahulukan kereta api; dan c. memberikan hak utama kepada Kendaraan yang lebih dahulu melintasi rel. - 64 - Kecepatan

## Catatan

Belum ada.
